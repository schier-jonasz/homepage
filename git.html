GIT vs. GITHUB

Git jest rozproszonym narzędziem kontroli wersji, które może zarządzać historią kodu źródłowego projektu rozwojowego, natomiast GitHub jest platformą opartą na chmurze zbudowaną wokół narzędzia Git. Git jest narzędziem, które programista instaluje lokalnie na swoim komputerze, natomiast GitHub jest serwisem internetowym, który przechowuje kod wypchnięty do niego z komputerów z uruchomionym narzędziem Git. Kluczowa różnica między Gitem a GitHubem polega na tym, że Git jest narzędziem open-source, które programiści instalują lokalnie, aby zarządzać kodem źródłowym, natomiast GitHub jest usługą online, do której programiści korzystający z Gita mogą łączyć się i przesyłać lub pobierać zasoby.
